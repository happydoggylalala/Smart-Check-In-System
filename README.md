# 智慧報到系統

純前端（無後端伺服器）的活動現場報到系統，資料儲存在瀏覽器 localStorage。

## 功能

- 名單匯入（CSV / Excel），自動偵測欄位並可手動對應
- 人工搜尋姓名/工號/Email 完成報到
- 報到/簽退狀態判定：
  - 活動開始 15 分鐘內報到 → **準時 ✓**
  - 活動開始 15–30 分鐘內報到 → **遲到 ⏰**
  - 活動開始 30 分鐘後 → **系統阻擋報到**，該人狀態顯示為**未到 ✕**
  - 活動結束前 30 分鐘內簽退 → **早退 ⚠**（疊加徽章，與報到狀態分開顯示）
- 現場候補：非名單人員現場報到時，檢查目前已報到人數是否達活動容量上限，未滿則加入名單並標記「現場候補」，已滿則阻擋並提示錯誤
- 早鳥福利：依「報名時間」排序，最早的前 N 名自動標記「早鳥」徽章（可在設定調整 N）
- 報到即入抽獎池：所有成功報到者自動列入抽獎資格，現場可一鍵抽獎，並可選擇排除已得獎者
- 智慧分組座位：依報到成功的先後順序，輪流分配到各組與座位號（可設定組數與每組人數）
- 報到後自動寄送講義 Email：透過 [EmailJS](https://www.emailjs.com/) 從瀏覽器直接寄信，寄信失敗不會卡住報到流程

## 啟動方式

因使用 ES modules（`<script type="module">`），**不能直接用瀏覽器開啟 `index.html`（file://）**，會被瀏覽器的模組載入 CORS 限制擋下，localStorage 行為在 file:// 下也不穩定。請用任一種免安裝的靜態伺服器啟動：

```bash
# 方法一：Node.js（若已安裝）
npx serve .

# 方法二：Python（若已安裝）
python -m http.server 8000
```

啟動後於瀏覽器開啟顯示的網址（例如 `http://localhost:3000` 或 `http://localhost:8000`）。

## 使用流程

1. **設定**頁：填寫活動名稱、開始/結束時間、容量、遲到/未到/早退門檻、早鳥名額、分組數與每組人數，儲存設定。
2. **名單匯入**頁：上傳 CSV/Excel（可先用專案內的 `sample-roster.csv` 測試），確認欄位對應後匯入。
3. **報到**頁：搜尋姓名/工號，點擊「報到」；找不到者可用「現場報到」表單登記（受人數上限檢查）。
4. **儀表板**頁：查看即時統計與完整名單，可匯出 CSV/Excel。
5. **分組座位**頁：查看各組配位情況，適合投影於現場。
6. **抽獎**頁：對已報到人員進行現場抽獎。

## EmailJS 設定（報到自動寄送講義）

1. 至 [emailjs.com](https://www.emailjs.com/) 免費註冊帳號，建立一組 Email Service 與 Email Template。
2. Email Template 建議包含下列變數：`to_email`、`to_name`、`event_name`、`handout_link`、`group_seat`。
3. 在系統的「設定」頁填入 Service ID、Template ID、Public Key 與講義連結，勾選「啟用報到後自動寄送講義 Email」。
4. 可用「測試寄送給自己」功能確認設定正確，再開放現場報到使用。
5. 若未設定或寄送失敗，報到流程仍會正常完成，僅該筆資料的 Email 狀態顯示為「未設定」或「失敗」，需事後在儀表板追蹤補寄。

## 重要設計取捨

- **單一裝置假設**：localStorage 僅存在單一瀏覽器/裝置，不會跨裝置同步。若需多站報到，可用「設定」頁的 JSON 備份匯出/匯入功能，在交接班或合併資料時手動轉移。
- **逾時報到直接阻擋**：超過活動開始 30 分鐘（可調整）後，系統不再允許報到，並將該人狀態視為「未到」——這是依需求明確選擇的規則，而非允許逾時報到但標記其他狀態。
- **座位超額不阻擋報到**：若報到人數超過原規劃的「組數 × 每組人數」，座位號會自然遞增超過原規劃值，UI 顯示「超出原座位規劃」提示，但不會阻擋報到；唯一會阻擋報到的規則是上述的報到時間關卡，以及現場候補的人數上限關卡。
- **Email 寄送非同步、不阻塞報到**：報到成功後立即完成狀態更新與畫面顯示，寄信是背景非同步進行，結果只更新該筆資料的 Email 狀態欄位。

## 專案結構

```
index.html          單頁應用外殼，載入 CDN 依賴與 js/app.js
css/style.css        版面與樣式
js/
  app.js              進入點，wiring 各畫面模組
  state.js             localStorage 讀寫（唯一存取點）
  models.js             資料模型工廠函式
  utils.js               共用工具函式
  importRoster.js         CSV/Excel 匯入
  checkin.js               報到/簽退核心邏輯與畫面
  waitlist.js               現場候補人數上限檢查
  earlybird.js               早鳥福利計算
  grouping.js                 智慧分組座位配位
  lottery.js                   抽獎邏輯與畫面
  email.js                      EmailJS 寄信
  dashboard.js                   儀表板統計與名單表格
  settingsScreen.js                設定頁
  exportData.js                     匯出 CSV/Excel
  nav.js                              分頁導覽
sample-roster.csv    測試用範例名單
```
